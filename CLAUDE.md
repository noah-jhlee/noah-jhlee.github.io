# CLAUDE.md

## 프로젝트 개요
이 프로젝트는 Astro 기반 개인 홈페이지이며, 최종 배포 목표는 GitHub Pages (github.io)다.

## 규칙

### 라이브러리 및 파일
- 새 라이브러리는 사용자 승인 없이 추가하지 않는다.
- 현재 작업과 관련 없는 파일은 수정하지 않는다.

### 보안
- secret, token, password, API key를 코드에 직접 작성하지 않는다.

### 명령 실행
- 삭제 명령(rm, git clean 등)은 실행 전에 반드시 사용자에게 확인한다.

### 변경사항 관리
- 수정 후 변경한 파일과 이유를 사용자에게 설명한다.
- 가능하면 `npm run build`로 빌드 오류 여부를 확인한다.
- 변경사항이 생기면 자주 commit한다.

### Commit 메시지 형식
- `feat: 한국어설명` — 새 기능 추가
- `fix: 한국어설명` — 버그 수정
- `chore: 한국어설명` — 설정, 패키지 등 기타 작업
- `docs: 한국어설명` — 문서 수정
